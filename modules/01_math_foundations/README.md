# Week 1 — Mathematical foundations

## Learning objectives

By the end of this week you can:

1. Interpret matrices as linear maps and use **SVD / eigendecomposition / pseudoinverse** fluently.
2. Manipulate **probability densities**, exponential families, and core information-theoretic quantities (entropy, KL, cross-entropy, mutual information).
3. Recognise **convex sets and functions**, set up Lagrangian duals, and reason about the convergence rates of gradient descent.
4. Explain the **Fokker–Planck / Langevin** picture well enough to understand diffusion models in Week 10.

## Topics

- Linear algebra: rank, four fundamental subspaces, orthogonal projections, SVD geometry, Moore–Penrose pseudoinverse, matrix calculus.
- Probability: measure-theoretic basics (skim), exponential families, KL divergence as log-likelihood-ratio expectation, Fisher information, cross-entropy.
- Optimisation: convex sets/functions, first-order conditions, duality, GD / SGD / momentum convergence rates, Polyak–Łojasiewicz.
- Stochastic dynamics: Brownian motion, Langevin SDE, Fokker–Planck equation (qualitative).
