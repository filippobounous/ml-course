# Week 4 — Classical unsupervised learning

## Learning objectives

1. View **PCA / SVD** simultaneously as variance maximisation, reconstruction-error minimisation, and MLE of probabilistic PCA.
2. Derive **k-means** as coordinate descent and **GMM / EM** as latent-variable MLE.
3. Reason about **density estimation** via KDE and foreshadow normalizing flows.
4. Apply PCA to a **statistical-arbitrage** problem on equity returns (quant-finance domain).

## Topics

- PCA: variance maximisation, SVD equivalence, probabilistic PCA, factor models.
- Clustering: k-means, k-means++, GMM with full / diagonal / spherical covariance.
- EM: derivation for GMM, connection to variational inference.
- Density estimation: histograms, KDE, bandwidth selection; preview of normalising flows.

## Deliverables

- Portfolio artifact: `portfolio/04_pca_statarb/` — PCA stat-arb notebook on Ken French industry portfolios; in-sample/out-of-sample Sharpe; write-up following Avellaneda & Lee (2008).
