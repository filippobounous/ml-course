> Populated in Week 2. See `modules/02_stat_learning/`.

## Problem
Build a clean, tested, pure-NumPy implementation of linear regression with
closed-form and SGD solvers, ridge and lasso regularisation, and K-fold
cross-validation.

## Method
- Normal equations (pseudo-inverse path for rank-deficient X).
- SGD with momentum.
- Lasso via coordinate descent.
- K-fold CV with stratification option.
